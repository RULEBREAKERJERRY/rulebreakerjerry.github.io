AI 入门笔记

Posted on 2025-06-13 In 学习笔记

一、深度学习框架（Deep Learning Frameworks）

1. PyTorch

  • 编程语言：Python
  • 特点：
    • 写法灵活，贴近 Python 语法
    • 动态图机制，调试更友好
    • 科研界主流框架，论文复现多

2. TensorFlow

  • 编程语言：Python
  • 特点：
    • 静态图结构，更适合部署
    • 工业级优化好，适合大规模应用
    • 可与 Keras 配合使用，提高开发效率

二、神经网络结构（Neural Network Architectures）

1. CNN（卷积神经网络）

  • 全称：Convolutional Neural Network
  • 主要用途：图像分类、目标检测、视觉识别
  • 特点：
    • 局部感受野 + 权重共享
    • 能提取图像中边缘、纹理等空间特征

2. RNN（循环神经网络）

  • 全称：Recurrent Neural Network
  • 主要用途：时间序列预测、语音识别、传统NLP任务
  • 特点：
    • 处理序列数据，前一个时刻会影响当前输出
    • 可建模时间依赖关系

3. Transformer

  • 主要用途：文本生成、图像描述、多模态任务
  • 特点：
    • 自注意力机制（Self-Attention）建模全局信息
    • 高度并行化，训练效率高
    • 是 GPT、BERT、DALL·E、Stable Diffusion 等模型的基础结构

三、生成模型（Generative Models）

1. VAE（变分自编码器）

  • 全称：Variational AutoEncoder
  • 原理：编码–解码结构中加入高斯噪声，实现多样性生成
  • 特点：
    • 训练稳定
    • 生成图像清晰度较低

2. GAN（生成对抗网络）

  • 全称：Generative Adversarial Network
  • 原理：生成器与判别器博弈，提升生成图像的真实度
  • 特点：
    • 生成图像非常逼真
    • 训练过程容易不稳定

3. Diffusion Model（扩散模型）

  • 原理：从噪声逐步反向恢复图像（去噪过程）
  • 特点：
    • 稳定性好
    • 生成质量高（主流AI绘画模型的核心）

四、知识层次结构图

人工智能（AI）
│
├── 机器学习（Machine Learning）
│   └── 深度学习（Deep Learning）

│       │
│       ├── 神经网络结构
│       │   ├── CNN（图像方向）
│       │   ├── RNN（序列方向）
│       │   └── Transformer
│       │
│       └── 生成模型（Generative Models）

│           └── Diffusion
