五、分支结构

1. 三目运算符

三目运算符的结构是：

  • 如果条件为真，就返回值1
  • 否则返回值2

典型用途场景：

场景 示例代码
变量赋值时按条件取值 max_val = a if a > b else b
一行返回函数结果 return "YES" if flag else "NO"
列表推导式中配合使用 [x if x % 2 == 0 else 0 for x in nums]

2. 判断容器是否为空：if not xxx:

类型 空值 示例 布尔值
列表 [] if not my_list: False
元组 () if not my_tuple: False
字典 {} if not my_dict: False
集合 set() if not my_set: False
字符串 "" if not my_str: False
range对象 range(0) if not my_range: False
